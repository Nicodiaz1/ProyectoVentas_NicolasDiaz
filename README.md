# Proyecto Ventas — Análisis y Dashboard

Limpieza, transformación y análisis de datos de ventas, con visualización
final en un dashboard de Power BI.

## Contenido
- `Avance1/` — limpieza y transformación (`ventas.csv` → `ventasTransformed.csv`) + modelo Power BI (`.pbix`).
- `Avance2/`, `Avance3/` — etapas siguientes del análisis.
- `Conclusiones_Recomendaciones.pdf` — hallazgos de negocio.

## Tecnologías
Power BI · Python / Jupyter · pandas
